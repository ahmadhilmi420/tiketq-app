## Features

- Live previews
- Fullscreen mode
- Cross platform
- Filtering (show only used or unused tickets).

## Setup and Run Locally

Clone the project

```bash
  git clone https://github.com/ahmadhilmi420/tiketq-app.git
```

Go to the project directory

```bash
  cd tiketq-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
